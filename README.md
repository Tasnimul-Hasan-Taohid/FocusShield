# 🛡️ FocusShield

FocusShield is a simple productivity web app that helps you stay focused by blocking distractions and tracking your focus time.

---

## 🚀 Features
- Add and manage distracting websites
- Focus timer (Pomodoro-style)
- Tracks total focus time
- Clean and minimal UI

---

## 🛠️ How to Use
1. Download or clone this repository
2. Open `index.html` in your browser
3. Add distracting sites (e.g., youtube.com)
4. Enter focus time and start the timer
5. Stay focused 💪
